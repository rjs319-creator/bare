# Experiment #5 — Historical-Twin Incremental Value (2026-07)

**Question.** Does NSL Engine 8 (counterfactual historical-twin estimation — a k-NN analog
predictor over pre-decision technical state) add forward predictive value **orthogonal to price
momentum**, on real free data?

**Status:** Demonstrated out of sample: NO edge (orthogonal IC ≤ 0)

## Method
- Universe: 56 liquid large/mid-caps — the SAME cross-section as experiments #3–#4, for comparability.
- Baseline: 6–1 price momentum (126-bar lookback, 5-bar skip), point-in-time.
- Signal: median forward outcome of a name's nearest historical **twins**, matched on 4 PIT features
  (6–1 momentum, 1-month reversal, 63d realized vol, % vs 200d SMA). Because `mom` is BOTH a feature
  and the baseline, the orthogonal IC isolates the twins' **nonlinear** analog structure beyond linear momentum.
- Twin pool: resolved analog states only (**labelEndDate ≤ decision date** — no outcome look-ahead); a name is **self-excluded** (never twins with its own history).
- Label: real next-open fill held 21 sessions (purged; unelapsed dates dropped).
- Panel: 17 non-overlapping monthly decision dates (2025-01-23 → 2026-05-27).
- Test: date-clustered rank-IC of baseline / augmented / signal-alone / **baseline-orthogonal** signal,
  Bonferroni over 3 specification variants (`lib/nsl/incremental.js`).

## Panel diagnostics
- Analog library: **3136** resolved states  ·  per-date resolved pool: **0–2744**
- Samples: **669**  ·  non-zero twin readings: **669**  ·  used dates: **12**
- Dropped — no features/momentum: 224, insufficient twin support: 59, label unelapsed: 0

## Result

| Model | Forward rank-IC |
|---|---|
| Baseline (momentum) | IC 0.0407, t 0.95, n 12 |
| Augmented (momentum + twin) | IC 0.04, t 1.11, n 12 |
| Twin alone | IC 0.0097, t 0.19, n 12 |
| **Twin ⟂ momentum (incremental)** | IC -0.0108, t -0.22, n 12 |

- ΔIC (augmented − baseline): **-0.0007**
- Incremental significant after Bonferroni (t ≥ 2): **false**
- **Verdict:** `no-edge` → recommendation `reject`

## Honest reading
The decisive quantity is the **baseline-orthogonal** twin IC. It is not positive, so on this panel the historical-twin analog adds no value beyond linear momentum after the false-discovery correction — an honest negative.

## Known limitations (not code-solvable on the free/serverless stack)
- Universe is **current survivors** (no delisted names) — survivorship-unsafe; a production verdict needs a PIT security master.
- Only 56 names over a 2-year window, so the analog pool is modest and drawn from one macro regime; twins from a different regime could behave differently.
- The 4-feature state space is deliberately small and price-only; a richer (fundamental / cross-asset) description could find sharper analogs — or overfit.
- Monthly labels are ~non-overlapping but adjacent; residual autocorrelation is possible.
